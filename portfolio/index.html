Universidade Lusófona de Humanidades E Tecnologia

Problema e Objetivo

A razão pela qual as organizações escrevem requisitos de baixa qualidade, é a falta de recursos adequados para o processo de análise e levantamento de requisitos. O atual projeto visa garantir a boa qualidade dos requisitos levantados pelas organizações e que os mesmos sejam úteis para os projetos , isso através de um uma plataforma WEB desenvolvida em contexto académico que avalia os requisitos dentro de um projeto.

Publico Alvo
Destinado a todas as pequena empresas ou organizações que optam por um produto únicamente direcionado aos requisitos.
O projeto em questão será utilizado no momento de recolha dos requisitos sendo assim utilizado pelos diferentes stakelholders involvidos no projeto correspondente.
