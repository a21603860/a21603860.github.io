Introdução

O presente trabalho é realizado no contexto académico de Trabalho de fim do curso(TFC) e nele foi possível mostrar o processo de gestão de requisitos de software.

Plataforma de Gestão de Requisitos

Problema e Objetivo

O atual projeto visa garantir a boa qualidade dos requisitos uma vez que os mesmos representam uma percentagem elevada quando o assunto é projetos falhados. Através desta plataforma as organizações poderão levantar, analisar e avaliar requisitos de forma a saber se o mesmo é útil para o projeto em questão.

Publico Alvo
Destinado a todas as pequena empresas ou organizações que optam por um produto únicamente direcionado aos requisitos.
Este diagrama representa diversos processos dentre os quais, Login, criação de requisitos , de projetos e aceder a lista de requisitos criados, isso na prespectiva do utilizador final.
